Setup

npm install -g ionic

Comandos básicos

Crear un Proyecto
ionic start todo blank
Configurar la plataforma
ionic platform add ios
ionic platform add android
Compilar
ionic build ios
ionic emulate ios

Creando Iconos

Salve su imagen como icon.png, icon.psd, icon.ai la dimensión mínima es 192x192px coloquelo en la carpeta root y ejecute el siguiente comando
ionic resources --icon

Splash Screen desde una Imagen

Salve la imagen en splash.png, splash.psd, splash.ai ponlo en el directorio resources la imagen debe tener un mínimo de 2208x2208px, y ejecute el siguiente comando
ionic resources --splash

Ejecutar tu aplicación

cd myApp
ionic serve

Creando Aplicación de IONIC

ionic start MyIonic2Project tutorial --v2

Tipos de plantillas de inicio de Ionic

Plantilla en Blanco
ionic start --v2 myApp blank
Plantilla con Tabs
ionic start --v2 myApp tabs
Plantilla con Menú
ionic start --v2 myApp sidemenu

Mandar a compilar APK

Una vez que creamos la aplicación la compilamos para distribuirla
cordova build --release android
